\f0\fs24 \cf0 # Merchant Risk Scanner \uc0\u55357 \u57057 \u65039 \
\
An automated Merchant Underwriting and Risk Scoring tool designed for **RBI-Licensed Payment Aggregators**. \
\
## Features\
- **Web Scraping:** Extracts text from merchant websites using BeautifulSoup.\
- **Weighted Risk Engine:** Scores merchants based on a library of high-risk keywords (Forex, Crypto, MLM, etc.).\
- **Audit Reports:** Generates a downloadable PDF report for compliance records.\
\
